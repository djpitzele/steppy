# steppy
Bitcamp 2024 project by CJ Nygard and Daniel Pitzele

Components:
- Arduino Uno
- Grove Shield
- Grove Button (2x) 
- Grove LED
- Grove LCD RGB Backlight

Incrementor (left) button in port D4.
Next (right) button in port D3.
LED in port D2.
LCD in port I2C.
